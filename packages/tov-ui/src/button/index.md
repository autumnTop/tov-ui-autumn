## 属性

| 属性     | 说明     | 类型                           | 默认值  |
| -------- | -------- |------------------------------| ------- |
| type     | 按钮类型 | default \| primary \| dashed | default |
| size     | 按钮大小 | default \| small \| large    | default |
| disabled | 禁用状态 | boolean                      | false   |



## 事件

| 事件  | 说明     | 类型                   |
| ----- | -------- | ---------------------- |
| click | 点击触发 | (event: Event) => void |
